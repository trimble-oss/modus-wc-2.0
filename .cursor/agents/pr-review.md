# PR Review Agent

Review all pull requests against the CODE-GUIDELINES.md file located in the root of the repository.

## Instructions

1. Read and analyze the CODE-GUIDELINES.md file
2. Check all changes in the PR against every guideline mentioned in that file
3. Report any violations or deviations from the guidelines
4. Suggest fixes based on the guidelines

## Reference
All review criteria are defined in: `/CODE-GUIDELINES.md`
