# Getting Started

## Installation

```bash
npm install @trimble-cms/modus-wc
```

## Usage

1. Register custom elements:

```javascript
import { defineCustomElements } from '@trimble-cms/modus-wc/loader';

// Call during the initial loading of your application
const Root = () => {
  defineCustomElements();

  return <App />;
};
```

2. Use the components:

```javascript
// Use in HTML
<modus-wc-button variant="primary">Click me</modus-wc-button>
```
