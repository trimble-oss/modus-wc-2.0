# Getting Started

## Installation

<b>
  Lock the installed package version to avoid unintended breakages on future npm
  installs.
</b>

```bash
npm install @trimble-cms/modus-wc
```

## Usage

### 1. Set the theme:

The theme can be set manually or by using the `ThemeSwitcher` component. See the "Use a Theme" section of [Custom Styling](/docs/documentation-custom-styling--docs) for guidance.

The default themes are:

- `modus-classic-light`
- `modus-classic-dark`

### 2. Register custom elements:

```javascript
import { defineCustomElements } from '@trimble-cms/modus-wc/loader';

// Call during the initial loading of your application
const Root = () => {
  defineCustomElements();

  return <App />;
};
```

### 3. Use the components:

```javascript
// Use in HTML
<modus-wc-button variant="primary">Click me</modus-wc-button>
```
