# Angular Workspace: ng18

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 18.2.9.

This Angular Project is a utility wrapper to help build the `@trimble-cms/modus-wc-ng` library.

All components in this repository are auto generated by the StencilJS library.

This library can be consumed in a local angular application using npm link`

## Building Locally

To rebuild the Modus Angular Components you need to perform the following steps:

- From the `./integrations/angular/ng18` project directory run
  `npm install` then `npm run build`

- If you want to reference a local build of `modus-wc` you can run `npm pack` from the `poc-modus-wc-2.0` root 
directory and then `npm install path/to/modus-wc-tarball.tgz` from `ng18` directory.
