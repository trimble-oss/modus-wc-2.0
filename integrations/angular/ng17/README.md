# Angular Workspace: ng17

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.3.11.

This Angular Project is a utility wrapper to help build the `@trimble-cms/modus-wc-angular` library.

All components in this repository are auto generated by the StencilJS library.

This library can be consumed in a local angular application using npm link`

## Build

To rebuild the Modus Angular Components you need to perform the following steps:

- From the `./integrations/angular/ng17` project directory run
  `npm install` then `npm run build`

### Referencing a local build of modus-wc

#### Using npm link

- If you want to reference a local build of `modus-wc` you can run `npm link` from the `poc-modus-wc-2.0` root 
directory and then `npm link @trimble-cms/modus-wc` from the `ng17` directory. 

#### Using npm pack

- You can also run `npm pack` from the `poc-modus-wc-2.0` root 
directory and then `npm install path/to/modus-wc-tarball.tgz` from `ng17` directory.
