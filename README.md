# Modus Web Components 2.0

This is a web component library built with Stencil JS and SASS for large enterprise applications, using the prefix "modus-wc".

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm start` to start the development server
4. Run `npm run storybook` to start Storybook

## Building

Run `npm run build` to build the component library.

## Testing

Run `npm test` to run the test suite.

## Usage

To use a component in your project, import it and use it with the "modus-wc" prefix:

```html
<modus-wc-button label="Click me"></modus-wc-button>
```

## Theming

You can customize the appearance of components by overriding the CSS custom properties. All properties are prefixed with "modus-wc-".
